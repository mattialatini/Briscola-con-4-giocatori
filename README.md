# Briscola-con-4-giocatori
Il progetto non è completo, non abbiamo ancora finito alcune classi e la progettazione.

Il progetto consiste nella realizzazione del gioco Briscola a coppie con 4 giocatori usando un’architettura client-server. Il server gestisce le regole, i turni e i punteggi, mentre i client permettono ai giocatori di inviare le mosse e vedere lo stato della partita.
Per eseguire il Client bisogna eseguire ClientBriscola.java, Per eseguire il Server bisogna eseguire ServerBriscola.java.
